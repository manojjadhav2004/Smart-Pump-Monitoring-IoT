# Smart Pump Monitoring System

An IoT-based smart pump monitoring and protection system using ESP32.

## Features

- Remote pump control
- Flow monitoring
- Dry run protection
- Electricity fault detection
- Starter temperature monitoring
- Blynk mobile dashboard
- HTML web dashboard

## Hardware Used

- ESP32
- Flow sensor
- DHT11 temperature sensor
- Relay module
- Motor starter

## Software Used

- Arduino IDE
- Blynk IoT
- HTML + JavaScript dashboard

## Dashboard

Displays:

- Motor status
- Flow rate
- Temperature
- Fault reason
